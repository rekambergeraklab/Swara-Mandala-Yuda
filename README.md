# SWARA MANDALA YUDA
**Pengendali Otomasi Panoramix (Spatial Audio Controller)**
*Yogyakarta-Indonesia, 2026*

## 📖 Deskripsi
**SWARA MANDALA YUDA** adalah sekuenser audio spasial 64-saluran mandiri yang dirancang untuk **IRCAM Panoramix**. Menggunakan protokol OSC untuk menghasilkan lintasan suara yang kompleks dan organik.

## ✨ Fitur Utama
* **5-Layer Architecture:** Mengontrol hingga 64 trek secara independen atau simultan.
* **Organic LFO:** Modulasi "napas" untuk pergerakan suara yang lebih hidup.
* **Morphing Transitions:** Transisi halus antar pola geometris.
* **Auto-Save:** Menyimpan status terakhir sesi Anda secara otomatis.

## 🌀 Glosarium Pola (Movement Patterns)
Aplikasi ini menggunakan terminologi lokal untuk mendeskripsikan pergerakan spasial:
* **Mandala:** Orbit lingkaran sempurna.
* **Sapit 8:** Pola angka delapan (infinity).
* **Riris Harda:** Pola hujan jatuh dari atas.
* **Awu-awu Langit:** Pola tornado/corong.
* **Nginter Gabah:** Pemindaian raster cepat.

## 🚀 Cara Menjalankan
Aplikasi ini tersedia untuk tiga sistem operasi utama (cek bagian **Releases/Artifacts** di GitHub Actions):
1. **Windows:** Jalankan `Swara_Mandala_Yuda.exe`.
2. **macOS:** Buka `Swara_Mandala_Yuda.dmg` dan geser ke Applications.
3. **Linux:** Jalankan file `AppImage` atau `Swara_Mandala_Yuda` binary.
